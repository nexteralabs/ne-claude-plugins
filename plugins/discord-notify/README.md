# Discord Notify Plugin

Send messages to Discord channels from Claude Code.

## Setup

```
/discord-setup
```

Walks you through creating a Discord bot, getting your token and server ID, and testing the connection.

Credentials are stored in `~/.claude/discord.env`.

## Usage

```
/discord general "Build passed, PR ready for review"
/discord pull-requests "KAN-42 — Fix login timeout — https://github.com/org/repo/pull/123"
```
