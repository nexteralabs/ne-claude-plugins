# Obsidian Sync Plugin

MCP server that connects Claude Code to your Obsidian vault for knowledge-base-driven development.

## Setup

Set the `OBSIDIAN_VAULT_PATH` environment variable to your vault location:

```bash
export OBSIDIAN_VAULT_PATH="$HOME/path/to/your/vault"
```

## Features

- Search and read notes from your Obsidian vault
- Create and update notes as part of development workflows
- Link project context to knowledge base entries
